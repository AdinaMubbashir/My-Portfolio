## Features

- 🕹️ **Arcade theme** — Pac-Man inspired design with neon purple, blue, and black palette
- 👻 **Playable Pac-Man** — fully functional game with ghosts, power pellets, lives, and a 3-2-1-GO countdown
- ✨ **Animated hero** — staggered fade-in entrance, animated grid background, floating pellets
- 📱 **Mobile friendly** — swipe to control the game, responsive layout throughout
- 🔦 **Neon glow effects** — scanline overlay, glowing maze walls, pulsing power pellets
- 🧭 **Smooth scroll nav** — fixed navbar with active section highlighting

## How to Deploy

**Netlify (recommended):**
- Set publish directory to `src`
- Leave build command blank
- Netlify will serve `src/index.html` directly

**Local:**
```bash
open src/index.html
```
No server needed — open the file directly in any browser.
